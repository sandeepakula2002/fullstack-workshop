MY PROJECT

PAYING GUEST MANAGEMENT SYSTEM

The PG Management System is a web-based application designed to simplify and automate the management of paying guest accommodations. It allows administrators to efficiently handle tenant details, room allocation, rent tracking, and basic record management through a user-friendly interface.

Developed using HTML, CSS, JavaScript, PHP, and MySQL, the project implements CRUD operations, form validation, and secure database interactions. This system improves accuracy, saves time for PG owners, and provides a reliable solution for managing daily PG operations effectively.
