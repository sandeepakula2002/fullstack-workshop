Project

Paying Guest Management System

The PG Management System is a web-based application developed to simplify and automate the management of paying guest accommodations. The system allows administrators to efficiently manage tenant details, room allocation, rent tracking, and basic record maintenance through a clean and user-friendly interface, reducing manual effort and improving accuracy.

This project is developed using HTML and CSS for structure and styling, JavaScript for client-side interactivity, PHP for server-side logic, and MySQL for database management. It implements core CRUD operations, proper form validation, and secure database interactions. Through this project, I gained hands-on experience in full-stack web development and learned how different technologies work together in a real-world application.
